Spelling : Adjective Superlatives

The girl is the nicest.     NOTE : If an adjective ends in e, add -st.
The girl is the prettiest.       If an adjective ends in y, change the y to i and add -est.
The girl is the thinnest.       If an adjective ends in one vowel and one consonant,
double the consonant before adding -est.
