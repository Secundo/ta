Spelling : Adjective Comparatives

Sam is heavier than Jeff.

Jeff is thinner than Sam.

NOTE :

If an adjective ends in e, add -r.

Add -er to most two-syllable adjectives ending in y.

If an adjective ends in y, change the y to i and add -er.

If an adjective ends in one vowel and one consonant,

double the consonant before adding -er : thinner. (Vowel consonant)
